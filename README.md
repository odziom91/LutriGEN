# LutriGEN
Install every Windows game in Lutris easly!  

## Requirements
- Python 3.x.x (tested on 3.9.2)  
- PySimpleGUI (included in modules directory)  
- Tkinter (tk) for Python  
- git (to clone this repository)  

## How to install
### Arch Linux
pacman -S python tk  
git clone https://github.com/odziom91/LutriGEN.git  
cd LutriGEN  
python lutrigen.py
